배경 이미지 테스트

background-image 속성을 이용하여 배경 사진을 넣을 수 있음. 
배경 이미지의 반복 방법을 지정하기 위해 background-repeat 속성을 이용할 수 있다.
배경 이미지가 요소보다 크거나 작은 경우 배경 이미지의 크기를 background-size속성으로 변경할 수 있다.
배경 이미지의 위치를 조정하기 위해서 background-position 속성을 사용할 수 있다.
배경 이미지가 움직이지 않게 고정하기 위해서 background-attachment 속성을 이용하여 웹페이지가 위아래로 움직여도 배경이미지는 움직이지 않게 할 수 있다.
